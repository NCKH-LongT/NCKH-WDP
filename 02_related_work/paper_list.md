## Main Research Focus

The selected papers focus on:

- AI-assisted source code evaluation,
- rubric-based grading,
- GitHub repository analytics,
- collaborative software project assessment,
- LLM-based grading systems,
- AI-human grading comparison.

These papers are highly relevant to the SEAL project because SEAL aims to integrate:

- GitHub repository tracking,
- AI-assisted code review,
- rubric-based evaluation,
- programming competition management.
