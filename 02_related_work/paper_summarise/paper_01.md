Problem

Bài báo giải quyết vấn đề:

sinh viên khó xác định nghề nghiệp phù hợp
thiếu hệ thống đánh giá kỹ năng cá nhân
thiếu định hướng kỹ năng theo ngành CNTT

Hệ thống hướng đến việc hỗ trợ sinh viên lựa chọn career path phù hợp dựa trên kỹ năng và dữ liệu học tập.

Method

Bài báo sử dụng:

Machine Learning
Skill assessment system
Network visualization
Career recommendation model

Hệ thống phân tích kỹ năng sinh viên và trực quan hóa mối liên hệ giữa:

skills
career roles
learning pathways
Dataset

Dữ liệu bao gồm:

hồ sơ kỹ năng sinh viên
dữ liệu học tập
thông tin nghề nghiệp ngành CNTT
dữ liệu mapping giữa kỹ năng và nghề nghiệp
Evaluation

Bài báo đánh giá thông qua:

accuracy của recommendation
khả năng matching giữa skill và career role
user feedback và usability evaluation
Results

Kết quả cho thấy:

hệ thống có thể hỗ trợ định hướng nghề nghiệp hiệu quả
cải thiện khả năng nhận diện skill gaps
giúp sinh viên hiểu rõ kỹ năng cần thiết cho từng nghề nghiệp
Limitations

Hạn chế:

dữ liệu chưa đủ lớn
chưa cập nhật theo xu hướng tuyển dụng thời gian thực
personalization chưa sâu
chưa tích hợp Large Language Models (LLMs)
