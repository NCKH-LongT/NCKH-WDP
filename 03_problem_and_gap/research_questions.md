# Research Questions

## Main Research Question

Can a MERN-based hackathon management system integrate GitHub automation and AI-assisted grading to rapidly evaluate source code submissions using dynamic rubrics with results aligned with human judges?

## Sub Research Questions

RQ1. Điểm AI theo từng rubric criterion có gần điểm judge không?

RQ2. Ranking do AI hỗ trợ có tương đồng với ranking của judge không?

RQ3. Tiêu chí nào AI đánh giá tốt hơn: functionality, code quality, documentation, efficiency, requirement compliance?

RQ4. Human-in-the-loop giúp giảm lỗi hoặc tăng độ tin cậy của AI scoring như thế nào?

RQ5. AI-assisted workflow giảm thời gian chấm bao nhiêu so với manual grading?

RQ6. GitHub API/Webhook có tự động hóa ổn định repo lifecycle không?
