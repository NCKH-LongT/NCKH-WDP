# Evaluation Metrics

| Metric | Meaning |
|---|---|
| MAE | Average difference between AI score and judge score |
| Pearson / R² | Score trend alignment |
| Spearman | Ranking similarity |
| Kendall's Tau | Pairwise ranking agreement |
| Top-k Agreement | Whether AI selects same top teams |
| Human Override Rate | Percentage of AI scores changed by judge |
| Time Reduction | Manual time vs AI-assisted time |
| Score Variance | Stability across repeated AI runs |
| Expert Rating | Judge usefulness rating |
| Webhook Capture Rate | GitHub event tracking reliability |
