# Diagrams

This folder contains Mermaid source files for diagrams. They can be rendered to PNG using Mermaid Live Editor or diagrams.net before final submission.

| File | Description |
|---|---|
| `architecture.mmd` | Overall MERN + GitHub + AI architecture |
| `iom_workflow.mmd` | Input–Output–Method workflow |
| `ai_evaluation_workflow.mmd` | AI evaluation pipeline |
| `judge_scoring_workflow.mmd` | Judge accept/override scoring workflow |

PNG files can be added later as:

```text
architecture.png
iom_workflow.png
ai_evaluation_workflow.png
judge_scoring_workflow.png
```
